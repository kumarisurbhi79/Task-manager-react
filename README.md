# Task Manager App

A modern, responsive task management application built with React and TypeScript.

## Features

- User Registration & Authentication
- Login/Logout functionality  
- Interactive Dashboard
- Add, Edit, Delete tasks
- Search and Filter tasks
- Responsive and attractive UI
- Local storage for data persistence

## Getting Started

1. Install dependencies:
```bash
npm install
```

2. Start development server:
```bash
npm run dev
```

3. Open your browser and navigate to the local development URL

## Usage

1. Register a new account or login with existing credentials
2. Use the dashboard to manage your tasks
3. Add new tasks with the "+" button
4. Edit tasks by clicking the edit icon
5. Delete tasks with the delete button
6. Use search and filter options to find specific tasks
7. Logout when done

## Tech Stack

- React 18
- TypeScript
- Vite
- CSS3 with modern styling
- Local Storage for data persistence
